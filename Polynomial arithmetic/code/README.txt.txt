单个C文件
文字编码：UTF8（部分编辑器如devc可能会出现乱码）

======Ver1.0=======
输入格式：
<多项式1>
Please input the operator: [+, *]
<多项式2>

多项式形式诸如`5x^3-3x^2+7x+8#`
必须按指数从高到低输入；
必须以#为结尾
数字和x之间有无*^等多余字符均可
但首字符必须是有效字符




======Ver2.0========
输入格式：
输入多项式的数量
逐个输入多项式，形式如Ver1.0
输入多项式间的算式，（以上多项式依次编号为1,2,3,……），可以使用括号，以等号为结束标志
	如："(1+2)*3="
